# Sawariya Diagnostic Website

**Sawariya Diagnostic** is a NABL-accredited pathology lab based in Charkhi Dadri, Haryana. This website serves as a digital storefront for patients to book tests, schedule home collections, and view health packages.

## üíª How to Run Locally

1.  **Clone the repository**
    ```bash
    git clone https://github.com/Sawariya-Diagnostic/Sawariya-Diagnostic.git
    cd Sawariya-Diagnostic
    ```

2.  **Install Dependencies**
    ```bash
    npm install
    ```

3.  **Start Development Server**
    ```bash
    npm run dev
    ```

4.  **Build for Production**
    ```bash
    npm run build
    ```
